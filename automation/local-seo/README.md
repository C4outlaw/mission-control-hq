# Local SEO Operating System (Claude Cowork)

This is the applied system for running repeatable local SEO audits and execution.

## Goal
Turn local SEO from random tasks into a weekly operating cadence with measurable outputs.

## Core Inputs (load once per business)
- Business profile: name, address, phone, website, GBP URL, service areas, target keywords
- Competitor profile list: business names + GBP URLs
- Current GBP export/details: categories, services, attributes, photos, posts, review metrics

## Weekly Cadence
1. **Week 1 (Foundation):** Category + Attributes audit
2. **Week 2 (On-Profile SEO):** Services + Description optimization
3. **Week 3 (Reputation Engine):** Review teardown + response system
4. **Week 4 (Content Velocity):** Posts + photos plan
5. **Repeat monthly** and compare with previous baseline

## Deliverables each cycle
- Gap report (what top competitors have that we do not)
- Prioritized action list (impact ranked)
- 8-week posting/photo queue
- Review velocity target + response templates
- Monthly scorecard (map pack visibility proxy, calls, direction requests, review growth)

## File Map
- `prompts.md` → production prompt stack (8 prompts)
- `client-template.md` → input template per business
- `execution-checklist.md` → operator checklist for weekly use

## Notes
- Prompts are designed for Chrome-based manual/assisted extraction.
- Keep all output in spreadsheets for comparison over time.
- Human decision-making remains required for prioritization and revenue alignment.
