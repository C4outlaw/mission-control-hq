# MyrieHQ V3 — Premium Asset Plan (Disruptive-Style Direction)

## Visual Direction
- Feel: premium, high-trust, conversion-focused, modern agency
- Mood: dark elegant base + electric blue accents + crisp white text
- Motion: subtle, intentional, smooth (no noisy animations)

## Required Assets

### Hero Video (1)
- Length: 8–12s loop
- Resolution: 1920x1080 (plus mobile-safe crop)
- Content: premium marketing/creative studio energy, restaurant/local business vibes
- No logos/watermarks

### Supporting Videos (3)
1. Website build/design montage
2. Restaurant/coffee brand visual refresh montage
3. Local business growth/dashboard montage

### Images (12–16)
- 4 service-related (web design, SEO, branding, automation)
- 4 hospitality-focused (restaurant/cafe interiors, plating, staff moments)
- 4 “results” style images (analytics, strategy desk, campaign planning)
- 2-4 optional city vibe images (Daytona/Orlando tone)

### Case Visuals (2)
- Beach Bucket project visual
- Magic Menu concept visual

## Optimization Targets
- Video: MP4/H.264, target <= 4MB for hero loop (desktop), <= 2MB mobile fallback
- Images: WebP/AVIF, target <= 250KB each for most cards
- Always include alt text and dimensions

## Build Integration Order
1. Ship structural V3 layout with placeholder media containers
2. Drop final curated assets
3. Run performance pass + lazy loading + preload hero
4. QA on mobile + desktop + Lighthouse
